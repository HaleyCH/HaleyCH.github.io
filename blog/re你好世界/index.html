原来的博客发生了什么？

长话短说，我不小心把之前博客的数据库删了。
就单纯是一不小心删除了了数据库文件。

这是什么？

这是我用 Hugo 搭建的博客，并使用 Github Actions 自动部署。

所有图片通过 七牛云 储存，本地上传采用了 PicGO。
全博客本着能用现成的就不造轮子的宗旨。

感觉比上一个博客舒服多了，具体流程我会在之后的文章中分享。

封面是什么？

封面是我无意中翻出去年自己设计的第一款鼠标垫的私货款式，因为不符合某些要求，最后没做出来。

总之，因祸得福

这是我的第六版博客。

自己手写了3版，第一版是Django乱做的，第二版是纯三件套搭的，第三版是用Vue3搭的。
第四版，也就是之前的博客，是用Hexo搭的，部署在自己服务器上，但是被我手抖删除了。
在这之后的半年里，我试着用wordpress搭建过博客，不尽如人意，然后我又用Vue3写了一个。
但是我太懒了，感觉也没提升，就搁置了。

现在的这一版，我放弃了自己手动搭博客，也懒得再在本地搭了。
托管在GitHub上，用现成的框架，用OSS储存图片，如果不是因为我第四版博客崩了，我也绝对不会接受这套方案。

现在感觉这套方案很好🥰。
